api_key : Your API Key

server_id: The server ID (Default value is 1 if not specified).

GET NUMBER

Request -> GET
https://api.otpbuddy.com/stubs/handler_api.php?api_key=$api_key&action=getNumber&service=$service&country=$country&server_id=$server_id

Parameters :

api_key : Your API Key

service : Service ID

country: The Country ID (Default value is 22 (India) if not specified).

server_id: The server ID (Default value is 1 if not specified).

Server Answer:

NO_NUMBERS : There are no numbers with the specified parameters, try again later, or change the country.

BAD_SERVICE : Incorrect service id

BAD_SERVER : Incorrect server id

BAD_COUNTRY : Incorrect Country id

NO_BALANCE : Run out of money for API-key

ERROR : Some Error Occured

RESPONSE 
{
    ACCESS_NUMBER:687d51e19c5297335bb10161:919067545744
}

GET SMS OR MESSAGE 

Request -> GET
https://api.otpbuddy.com/stubs/handler_api.php?api_key=$api_key&action=getStatus&id=$id

Parameters :

api_key : Your API Key

id : Activation ID
Server Answer:

STATUS_WAIT_CODE : Waiting for SMS

STATUS_CANCEL : Activation canceled

STATUS_OK:CODE : Code received (where CODE - activation code)

ERROR : Some Error Occured|
